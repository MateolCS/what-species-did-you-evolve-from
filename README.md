# What Species Did You Evolve From?

### Project Description

This project implements a rule-based expert system using CLIPS. The system performs reasoning through a dialogue with the user, following the principles of contextual data processing and production rules. Created as a part of Artificial Intelligence class at Poznan University of Technology.

## Key features
    1. Simple GUI made with Java Swing
    2. Dynamic reasoning - result is generated based on users choices during reasoning,
     instead of being stored locally

## Requirements
    1. JDK 17
    2. CLIPSJNI version 0.1 
    3. CLIPS version 6.30
